# Custom RISC-V AI Processor

## Overview

This project implements a custom 32-bit RISC-V processor featuring a 5-stage pipelined architecture and AI-oriented instruction extensions.

The processor includes hazard handling mechanisms, forwarding logic, and custom instructions for accelerating machine learning workloads.

---

## Features

- 32-bit RISC-V ISA
- 5-stage pipeline
- Forwarding Unit
- Hazard Detection Unit
- Custom AI Instructions
- Synthesizable RTL
- FPGA Ready Architecture

---

## Pipeline Stages

1. Instruction Fetch (IF)
2. Instruction Decode (ID)
3. Execute (EX)
4. Memory Access (MEM)
5. Write Back (WB)
